# Install: #

  - Install plugins.
    * Open Command Line in project folder and execute «npm install».
  - Initial Bower.
    * «bower init».
  - Make dir.
    * Run «mkdir.bat».
  - Insert TinyPNG key on tinypng-task.
  - Install 3rd-party plugins with command «bower install jquery -S».

# Folders structure: #

  - dev
    - blocks
      * _mixins
    - fonts (ttf fonts)
    - images
      * icons (SVG-icons for sprite)
    - js
    - pages
    - styles
      * helpers
  - public
    - css
    - fonts
    - images
    - js
